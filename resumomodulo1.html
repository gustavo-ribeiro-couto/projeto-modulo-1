  • o html é o componente base da web,isso quer dizer que ele permite a contrução de websites e a inserção de novos conteúdos,como imagems e vídeos,por meios dos hipertextos.
  • o HTML é o componente básico da web,ele permite inserir o conteúdo e estabelecer a estrutura básica de um website.portanto,ele serve para dar significado e organizar as imformações de uma pagina na web.sem isso,o navegador não saberia exibir textos como elementos ou carregar imagens e outros conteúdos.

    • sobre CSS

    CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML, por exemplo. De forma prática, ela funciona como uma camada de personalização ao conteúdo visível.
